# 小弛互联网科技公司网站

这是小弛互联网科技公司的官方网站源代码，一个简洁、响应式的企业展示网站。

## 功能特点

- 响应式设计，适配各种设备屏幕
- 简洁现代的UI设计
- 包含公司介绍、招聘信息和联系方式等核心内容
- 优化的页面加载速度

## 技术栈

- HTML5
- CSS3 (使用Flexbox进行布局)
- 纯静态网站，无需后端支持

## 本地开发

克隆此仓库后，您可以直接在浏览器中打开`index.html`文件查看网站效果。

```bash
git clone https://github.com/yourusername/xchhlw-website.git
cd xchhlw-website
```

## 部署

此项目已配置为可以直接部署到Vercel平台。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fyourusername%2Fxchhlw-website)
